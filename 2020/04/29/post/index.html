Everything Can RSS

Author: Luna Xu Date: April 29, 2020  11:23:04 Category: development

I’m trying to reduce the applicatoins I install in my phone, especially those social network applications. RSS is a good way to subscribe multiple sources in one application. But not everything has a rss service. Then I found this node.js project RSSHub to make everything RSSable! This is awesome and they have a lot of existing websites. However, they don’t have some website I want to add. So I contributed, see: Uniqlo Stylingbook, YAY! I will post more if I contribute more to this project. Stay tuned.
